# Return and Refund Policy

## Return Eligibility
Books can be returned within 7 days of delivery if:
- The book is in original condition (unread, no damage)
- Original packaging is intact
- You have the original invoice or Order ID

## Non-Returnable Items
- Books that have been read or have visible wear
- Digital books or eBooks
- Books purchased during clearance sales

## How to Initiate a Return
1. Go to "My Orders"
2. Select the order containing the book you want to return
3. Click "Return Item"
4. Select reason for return
5. Choose pickup date

## Refund Process
- Refunds are processed within 5-7 business days after we receive the returned book
- Refund is credited to your original payment method
- You will receive an email confirmation when refund is processed

## Exchange Policy
Instead of a refund, you can exchange for another book of equal or lesser value.
